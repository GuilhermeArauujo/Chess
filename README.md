# LAN Chess (2 Players)

A real-time 2-player chess game with full standard chess setup and rules:

- 8x8 board with full standard starting position
- White and Black piece sets (king, queen, rooks, bishops, knights, pawns)
- Legal move validation
- Check, checkmate, stalemate, draw detection
- Castling, en passant, pawn promotion
- Board orientation automatically flips to the active turn after each move

## Run locally

```bash
npm install
npm start
```

Then open:

- `http://localhost:3000` on two devices in the same LAN (or two browser tabs)
- Enter the same room ID on both clients
- First joiner is White, second joiner is Black
